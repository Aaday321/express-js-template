# Express.js Template

A minimal Express.js starter with a single `index.js` file.

## Requirements
- Node
- npm

## Setup
Install dependencies:

    npm install

## Start the server:

    node ./index.js
or

    node .

## Ensure its working
Once running, hit:

    http://localhost:3333

You should see:

    Hello World!
